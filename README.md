## Deployment from GitHub

To set up the project on another computer using GitHub:

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/your-repo.git
   cd your-repo
   ```

2. Install the dependencies:

   ```bash
   npm install
   ```

3. Start the development server:

   ```bash
   npx expo start
   ```

Make sure to replace `https://github.com/your-username/your-repo.git` with the actual repository URL.
