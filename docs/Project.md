# Проект: Мобильное приложение интервального таймера для спорта

## Цель проекта
Создать мобильное приложение для тренировок с интервальным таймером, обеспечивающее:
- Простой ввод пользовательских данных
- Гибкую настройку таймеров
- Надежную работу звуковых уведомлений
- Масштабируемую архитектуру для будущих доработок

## Архитектура
### Слой представления
- Flutter Widgets с использованием Bloc-паттерна для управления состоянием
- Многослойная структура экранов (4 основных экрана + модальные окна)

### Слой бизнес-логики
- Модуль управления таймерами (TimerManager)
- Система управления звуковыми уведомлениями (AudioNotifier)
- Валидация пользовательских данных

### Слой данных
- Hive DB для локального хранения профилей и таймеров
- JSON-сериализация конфигураций

## Этапы разработки
1. **MVP версия** (2 недели)
   - Реализация базовых экранов
   - Основная логика таймера
   - Локальное хранение данных

2. **Расширенная функциональность** (1 неделя)
   - Настройки звуков
   - Анимации интерфейса
   - Обработка ошибок

3. **Оптимизация** (3 дня)
   - Профилирование производительности
   - Юнит-тестирование

## Технологический стек
- ОС: Android/iOS (кроссплатформенная разработка)
- Язык: TypeScript
- Фреймворк: React Native
- State Management: Redux Toolkit
- Хранение данных: AsyncStorage + SQLite (react-native-sqlite-storage)
- Аудио: react-native-soun
- Навигация: React Navigation

## Стандарты
- **Консистентность**: Единый подход к навигации через NavigationService
- **Поддерживаемость**: Типизация через TypeScript, 70% покрытие тестами
- **Безопасность**: Шифрование данных в AsyncStorage через react-native-sensitive-info
- **Производительность**: Memoization компонентов, виртуализация списков
